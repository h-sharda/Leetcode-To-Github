Leetcode To GitHub Settings

To use this extension, you need a GitHub access token with permissions: Contents (Read and Write).

You can use an existing token or create a new one by following these steps:

  1. Visit GitHub Token Settings .
  2. Click Generate new token.
  3. Give it a descriptive name.
  4. Select Only select repositories under Repository Access.
  5. Choose your target repository from the dropdown.
  6. Under Repository permissions, go to Contents.
  7. Set it to Read and Write.
  8. Click Generate token.
  9. Copy the token and paste it below.

To fetch the details dynamically, use the following inputs:

Placeholder Description
<DD-MM-YYYY> gets date in specified format
<DD-MM> gets date in specified format
<MM-YYYY> gets date in specified format
<MM-YY> gets date in specified format
<DD> gets date in specified format
<URL> gets the URL of the problem
<PROBLEM_NAME> gets the problem name
<PROBLEM_NO> gets the problem number
<DIFFICULTY> difficulty of the problem
<CODE> your code fetched from the code editor
